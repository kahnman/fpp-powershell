# Get-FPPWledPalette

## SYNOPSIS
List available WLED palette names.

## SYNTAX

```
Get-FPPWledPalette [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
List available WLED palette names.

## EXAMPLES

### EXAMPLE 1
```
Get-FPPWledPalette
Returns a list of available WLED palette names.
```
