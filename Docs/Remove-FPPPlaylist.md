# Remove-FPPPlaylist

## SYNOPSIS
Delete a playlist from FPP.

## SYNTAX

```
Remove-FPPPlaylist [-Name] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Delete a playlist from FPP.

## EXAMPLES

### EXAMPLE 1
```
Remove-FPPPlaylist -Name 'Christmas'
Delete the playlist called "Christmas"
```
